## Source code

### Go
```go
package main

import "fmt"

func main() {
	for k := 10; k >= 1; k-- {
		fmt.Println(k)
	}
}
```
### Java
```java
public class SameProgram {
  public static void main(String [] args) {
    for (int k = 10; k >= 1; k--) {
      System.out.println(k);
    }
  }
}
```
